About VibeScout

VibeScout is a "vibe-based" AI agent built for the Vibe Coding competition. It uses Google's Gemini API to generate prototype "hidden gem" locations, complete with *dummy user reviews*, based on a user's mood.

This entire app, including the multi-page navigation and custom cursor, runs from a single HTML file.
